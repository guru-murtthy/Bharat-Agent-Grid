# Wireframe Notes

Open `docs/wireframes/index.html` in a browser. Use the browser’s Print → Save as
PDF (or screenshot each phone frame) to drop into the pitch deck.

## Six screens and what each proves to judges

1. **Home (mic-first)** — one big button, three language pills. Proves the
   low-literacy, voice-first thesis (Challenge 3).
2. **Plan + grounding** — shows the planner decomposition and that real sources
   were found. Proves the agent grid + grounding (Challenge 1).
3. **Verifier + consent** — APPROVED badge plus a spoken yes/no. Proves the
   consent gate before any real action (Trust Ledger primitive).
4. **Result + undo** — completed actions, an Undo button, and a ledger chain-OK
   line. Proves reversibility and auditability.
5. **Blocked** — the Verifier refusing an ungrounded action. Proves
   action-grounded anti-hallucination, the safety novelty (Challenge 1).
6. **Proof-of-Help** — a verified, shareable card with a referral code. Proves
   the trust-as-growth flywheel (Challenge 2).
